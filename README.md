## 🎯 Learning Objectives & Milestones
Inside this repository, we are systematically tracking our progress across four major frontend pillars:

- [x] **HTML5 Foundations:** Semantic layout structuring (`<header>`, `<main>`, `<footer>`).
- [x] **Media Handling:** Seamlessly embedding, sizing, and optimizing images.
- [x] **Interactive Forms:** Building user input forms, handling types, validations, and structures.
- [x] **Modern Frameworks:** Transitioning layouts into **Bootstrap** components and rapid styling using **Tailwind CSS**.

## 🛠️ How to View This Project Locally
To run and explore our code updates on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
2. **Navigate into the folder:**
   ```bash
   cd YOUR_REPOSITORY_NAME
   ```
3. **Open in VS Code:**
   ```bash
   code .
   ```
4. **Launch:** Open `index.html` directly in any web browser, or use the VS Code **Live Server** extension.

## 🤝 Collaboration Workflow Rules
To prevent code conflicts while we learn together, we follow these basic rules:
1. Always run `git pull` before starting a coding session to catch your partner's changes.
2. Commit frequently with clean descriptive messages (e.g., `git commit -m "feat: added login form validation"`).
3. Push variations in organized sub-folders to avoid overwriting each other's work files.
